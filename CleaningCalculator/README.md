# Калькулятор клінінга — адаптивний PWA

Це простий адаптивний веб-додаток для менеджера, який дозволяє:

- створювати та редагувати послуги
- зберігати список послуг локально в браузері
- додавати вибрані послуги до розрахунку
- обчислювати загальну суму
- встановлювати додаток як PWA

## Як запустити

1. Відкрийте `index.html` у браузері.
2. На смартфоні або в браузері підтримується встановлення з піктограми або діалогового вікна.

## Файли

- `index.html` — інтерфейс додатку
- `styles.css` — адаптивні стилі
- `app.js` — логіка калькулятора та збереження
- `manifest.json` — PWA-опис
- `service-worker.js` — кешування офлайн
- `icons/icon.svg` — значок додатку
